GitHub Authentication (Linux)

One-time setup:

    git config --global credential.helper store

First authenticated push:

    git push

When prompted:

    Username: TomWetmore
    Password: <GitHub Personal Access Token>

Git stores the token in ~/.git-credentials.

When the PAT expires:

    1. Regenerate/create a new PAT on GitHub.
    2. git push
    3. Enter the new token when prompted.
